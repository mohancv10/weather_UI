# Getting Started React App

This Application provides weather details for given location for last 7 seven days.

## Available Scripts

In the project directory, you can run:

After downloading the project kindly follow below steps
### `npm install`
### `npm start`

Runs the app in the development mode.
# Open [http://localhost:3000] to view it in your browser.

Steps:
### `Search and select locations from the dropdown.`
### `Selected Location will listed below.`
### `Click Search to calculate the Delta, Celsius and Fahrenheit values.`
### `Click Clear to reset the application to its initial value.`
